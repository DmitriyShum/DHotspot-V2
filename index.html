Full Sensor Irrigation (FSI)

This project was my first self-built Embedded Systems project which was built on an Arduino UNO in C++. Using two sensors, it is able to read the air temperature and humidity, as well as soil moisture, and write it to the 0.96 LCD. If the soil moisture reaches a certain high point level, that means that the soil is dry, the pump relay will turn on, and send water from the source, and into the soil.
